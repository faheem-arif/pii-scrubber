# Security Policy

## Reporting a Vulnerability
If you discover a security issue, please report it responsibly.

- Preferred: open a GitHub Security Advisory for this repository.
- If advisories are not available, contact the maintainer via their GitHub profile.
- Do not open public issues with sensitive details.

Please include:
- A clear description of the issue and potential impact.
- Steps to reproduce (with minimal data).
- Affected versions or commit hashes, if known.

We will acknowledge receipt when possible and work on a fix.
